⟨ WHY THIS EXISTS ⟩
Some domains use catch-all (wildcard) DNS records: every random subdomain resolves to the same IP. SpiderFoot's enumerator can return thousands of these "subdomains" which then create fake "shared infrastructure" signals in the main analysis. This tool detects wildcard zones so you can filter them out.
