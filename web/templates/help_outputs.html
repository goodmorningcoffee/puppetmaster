Output Files Explained

Each Puppet Analysis run creates a timestamped output directory with these files:

FileWhat it contains
executive_summary.md The main human-readable findings report. Start here. Lists smoking gun connections, top clusters, hub domains, and overall statistics.
smoking_guns.csv All confirmed connections (one per row). Shows the two domains, the signal type (e.g., google_analytics), and the shared value.
clusters.csv Domain clusters identified by Louvain community detection. One row per cluster with cluster ID, confidence, hub domain, and member list.
hub_analysis.csv Potential C2/controller domains ranked by network centrality (betweenness, PageRank). The "is_potential_c2" column flags likely controllers.
all_connections.csv Complete edge list with confidence and evidence summary for every domain pair that shares any signal.
signals.csv Raw signal table — every extracted signal with its type, value, and domain count. Useful for diagnosing flood filter behavior.
network.graphml The full graph in GraphML format. Open in Gephi or Cytoscape for interactive visualization and exploration.
